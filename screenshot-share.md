# Live Screenshot Sharing Instructions

## For Live Gameplay Sharing:

1. **Start the game** and click **"🔴 Live Share"** button
2. **Choose a save location** when prompted (e.g., Desktop/hot-gameplay-live.png)
3. The game will save screenshots to that location every 3 seconds
4. A red "LIVE Recording" indicator will appear

## How to Share with Claude:

After starting Live Share:
1. Play the game for a bit (at least 10-20 seconds)
2. Tell me: "Check the screenshot at [your save location]"
3. I can then use the Read tool to see your latest screenshot

Example:
```
You: "I saved the live screenshot to Desktop/hot-gameplay-live.png"
Claude: *uses Read tool to view the screenshot*
```

## Keyboard Shortcut:
- **Ctrl+Shift+L** - Toggle live recording on/off

## What I Can See:
- The full game screen
- Current year, marks, ships, reputation
- The phase you're in (House/Routes)
- The current event text
- All UI elements and choices

## Note:
The File System Access API used for live sharing works best in Chrome/Edge. Safari may have limitations.
