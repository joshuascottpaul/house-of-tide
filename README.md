# House of Tide

A narrative merchant dynasty game set in the archipelago of Verantia. You are the founder of a merchant house. You will trade, govern, and scheme. You will age. When you die, your heir continues.

## Features

- **Generational gameplay** — the campaign doesn't end when you die
- **AI-generated narrative** — dynamic events using Ollama (local) or Claude API
- **Junior gothic prose** — inspired by Hilary Mantel, Patrick O'Brian, Umberto Eco
- **Persistent consequences** — rivals remember, debts compound, threads stay open
- **Economic systems** — loans, ship market, passive fleet income
- **Character-driven heirs** — eight traits that affect both narrative and mechanics

## Play

Open `house-of-tide.html` in a browser. No server required.

**Requirements:**
- For Ollama: [Ollama app](https://ollama.com) running locally with a model pulled (`ollama pull mistral:latest`)
- For Claude: API key from [console.anthropic.com](https://console.anthropic.com)

Configure backend in ⊞ Settings on the title screen.

## The World

Verantia — the old city, built on reclaimed marsh. The council chamber leans slightly east. No one mentions this.

Three rival families:
- **The Borracchi** — old money performing new generosity
- **The Spinetta** — lost the silk route five years ago, still distributing blame
- **The Calmari** — very old, very quiet, leave council meetings with what they came for

The sea keeps better records than any notary. It simply does not share them.
